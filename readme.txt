== Description ==

A plugin to manage a collection of books with custom post type, shortcode, widget, and REST API endpoint.

== Installation ==

1. Upload the `books-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Books Settings' under 'Settings' to configure the plugin.

== Usage ==

1. Add books from the 'Books' menu in the WordPress admin.
2. Use the `[display_books]` shortcode to display the list of books on a page.
3. Use the 'Recent Books' widget to display the most recent books in a sidebar.
4. Access the REST API endpoint at `/wp-json/books/v1/list` to fetch the list of books.
